    Render(Graphics, Map)

    Draw the map decoration.

    Note that this base class' implementation resets g.Transform prior to raising event OnRendering, and restore the g.Transform prior to raising event OnRendered.

    Likewise, g.Clip is reset prior to rendering map decoration, and restored immediately after rendering.
