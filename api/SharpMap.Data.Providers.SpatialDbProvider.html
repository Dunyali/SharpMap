    GeometryExpression

    Expression template for geometry column evaluation.

    Declaration
    public string GeometryExpression { get; set; }
    Property Value
    Type Description
    String
    Examples

    You could, for instance, simplify your geometries before they're displayed. Simplification helps to speed the rendering of big geometries. Here's a sample code to simplify geometries using 100 meters of threshold.

    datasource.GeometryExpression = "ST.Simplify({0}, 100)";

    Also you could draw a 20 meters buffer around those little points:

    datasource.GeometryExpression = "ST.Buffer({0}, 20)";
