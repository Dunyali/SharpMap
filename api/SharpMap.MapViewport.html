    Envelope

    Gets the rectilinear extents of the map based on the current Zoom, Center, map Size, and (optionally) the MapTransform

    If a MapTransform is applied, the envelope CONTAINING the rotated view will be returned (used by layers to spatially select data) and the aspect ratio will NOT be the same as map Size. If aspect ratio is important then refer to Zoom and MapHeight
